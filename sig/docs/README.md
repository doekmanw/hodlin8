## SIG组工作目标和范围

### 工作目标
 为SIG仓提供文档、会议纪要及其他文档提供仓库，同时作为各SIG组的索引页

### 工作范围
* 存放各SIG组每次例会后产生的会议纪要、会议材料
* 提供各SIG组链接索引
* 为社区宣传提供可用的素材
