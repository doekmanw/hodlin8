# SIG-Meeting
简体中文 | [English](./sig_template.md)

说明：本SIG的内容遵循OpenHarmony的PMC管理章程 [README](/zh/pmc.md)中描述的约定。

## SIG组工作目标和范围

### 工作目标
 本SIG旨在为各SIG组的日常例会提供一个公开透明的展现平台。

### 工作范围
* 搜集各SIG组每次例会后产生的会议纪要、会议材料。
* 为社区宣传提供可用的素材。
